## Contributing Maintainers

Maintainers are contributors who have merged 5+ substantive PRs and have demonstrated:

- Understanding of the codebase and architecture
- Commitment to code quality and testing
- Respectful collaboration with other contributors
- Adherence to the Code of Conduct

### Becoming a Maintainer

Contributors interested in becoming maintainers can discuss this with the lead maintainer via GitHub issue or email.
